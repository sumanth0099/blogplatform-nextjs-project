## The Evolution of CSS

CSS has come a long way from simple styling. Modern CSS offers powerful features that make creating beautiful, responsive designs easier than ever.

### CSS Grid Layout

Create complex layouts with ease:
``````css
.container {
  display: grid;
  grid-template-columns: repeat(3, 1fr);
  gap: 2rem;
}

@media (max-width: 768px) {
  .container {
    grid-template-columns: 1fr;
  }
}
``````

### Flexbox

Perfect for one-dimensional layouts:
``````css
.flex-container {
  display: flex;
  justify-content: space-between;
  align-items: center;
  gap: 1rem;
}

.flex-item {
  flex: 1;
}
``````

## CSS Variables

Create maintainable stylesheets with custom properties:
``````css
:root {
  --primary-color: #3498db;
  --secondary-color: #2ecc71;
  --spacing-unit: 8px;
  --border-radius: 4px;
}

.button {
  background-color: var(--primary-color);
  padding: calc(var(--spacing-unit) * 2);
  border-radius: var(--border-radius);
}

/* Dark mode */
[data-theme="dark"] {
  --primary-color: #2980b9;
  --background: #1a1a1a;
  --text: #ffffff;
}
``````

## Modern Selectors

### :is() and :where()

Simplify complex selectors:
``````css
/* Instead of this */
.header a:hover,
.footer a:hover,
.sidebar a:hover {
  color: blue;
}

/* Use this */
:is(.header, .footer, .sidebar) a:hover {
  color: blue;
}
``````

### :has()

Style parent elements based on children:
``````css
/* Style cards that contain images */
.card:has(img) {
  display: grid;
  grid-template-columns: 1fr 2fr;
}
``````

## Container Queries

Responsive components, not just pages:
``````css
.card-container {
  container-type: inline-size;
}

@container (min-width: 400px) {
  .card {
    display: grid;
    grid-template-columns: 1fr 2fr;
  }
}
``````

## Advanced Animations

### CSS Transitions

Smooth state changes:
``````css
.button {
  background-color: blue;
  transition: all 0.3s ease;
}

.button:hover {
  background-color: darkblue;
  transform: translateY(-2px);
  box-shadow: 0 4px 8px rgba(0, 0, 0, 0.2);
}
``````

### CSS Animations

Create keyframe animations:
``````css
@keyframes slideIn {
  from {
    opacity: 0;
    transform: translateX(-100%);
  }
  to {
    opacity: 1;
    transform: translateX(0);
  }
}

.element {
  animation: slideIn 0.5s ease-out;
}
``````

## Clamp() Function

Fluid typography and spacing:
``````css
.title {
  /* Minimum 1.5rem, preferred 5vw, maximum 3rem */
  font-size: clamp(1.5rem, 5vw, 3rem);
}

.container {
  /* Responsive padding */
  padding: clamp(1rem, 3vw, 3rem);
}
``````

## Aspect Ratio

Maintain proportions without padding hacks:
``````css
.video-container {
  aspect-ratio: 16 / 9;
  width: 100%;
}

.square-box {
  aspect-ratio: 1;
}
``````

## Scroll Snap

Create smooth scrolling experiences:
``````css
.scroll-container {
  scroll-snap-type: x mandatory;
  overflow-x: scroll;
  display: flex;
}

.scroll-item {
  scroll-snap-align: start;
  flex-shrink: 0;
  width: 100%;
}
``````

## CSS Filters and Backdrop Filters

### Filters

Apply visual effects:
``````css
.image {
  filter: grayscale(50%) blur(2px);
}

.image:hover {
  filter: none;
  transition: filter 0.3s;
}
``````

### Backdrop Filters

Blur backgrounds:
``````css
.glass-card {
  background: rgba(255, 255, 255, 0.1);
  backdrop-filter: blur(10px);
  border-radius: 10px;
}
``````

## Logical Properties

Write direction-agnostic CSS:
``````css
/* Instead of margin-left */
.element {
  margin-inline-start: 1rem;
  padding-block: 2rem;
  border-inline: 1px solid black;
}
``````

## Best Practices

1. **Use CSS Variables**: Make your styles maintainable
2. **Mobile First**: Start with mobile styles, then enhance
3. **Semantic HTML**: Use appropriate HTML elements
4. **Accessibility**: Ensure good color contrast and focus states
5. **Performance**: Minimize repaints and reflows

## Browser Support

Always check browser support:
``````css
/* Feature detection with @supports */
@supports (display: grid) {
  .container {
    display: grid;
  }
}

@supports not (display: grid) {
  .container {
    display: flex;
  }
}
``````

## Conclusion

Modern CSS provides powerful tools for creating beautiful, responsive, and maintainable websites. By mastering these techniques, you can build better user experiences with less code.
